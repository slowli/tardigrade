Struct tardigrade_rt::storage::StreamingTransaction

source · [−]
pub struct StreamingTransaction<T> { /* private fields */ }
Expand description

Wrapper around a transaction that streams MessageEvents on commit. Used as the transaction type for Streaming storages.

Note that committing this transaction may involve dealing with backpressure.
