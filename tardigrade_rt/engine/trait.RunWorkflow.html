Expand description

Workflow instance execution logic.

A workflow instance wraps WorkflowData and can use its methods to manipulate the workflow in response to calls in the workflow logic (e.g., polling a next item for a channel receiver). Thus, WorkflowData methods conceptually map to WASM module imports. Conversely, methods in the RunWorkflow trait conceptually map to WASM module exports.
