Struct tardigrade_worker::FnHandler

pub struct FnHandler<F, Req, C = Json> { /* private fields */ }
Expand description

Worker handler based on a function. Mostly useful for quick prototyping and testing.

Examples

fn uses_handler(handler: impl WorkerInterface) {
    // snipped
}

let handler = FnHandler::json(|req: String| async move {
    req.repeat(2)
});
uses_handler(handler);
